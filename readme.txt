Easily display product barcodes in Worpress postings

== Description ==

The plugin adds a shortcode so you can embed barcode images into your postings, linking
to the barcode description on ean-search.org.

The plugin supports EAN, GTIN and UPC barcodes.

== Installation ==

1. Simply upload the file ean-barcode.php to your `/wp-content/plugins/` directory, or install the plugin or let WordPress install it there directly.
1. Activate the plugin through the 'Plugins' screen in WordPress


== Example ==

To display the barcode image for your MacBook Air, simply add this to your post:

MacBook Air `[barcode ean='00885909943074']`
